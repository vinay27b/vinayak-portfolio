# Vinayak Mahadeo Buchade – Personal Portfolio

## 🚀 Project Overview
A fully responsive, modern Personal Portfolio Web Application built with HTML5, CSS3, JavaScript, jQuery, and AJAX.

## 📁 Project Structure
```
portfolio/
├── index.html          # Main single-page portfolio
├── css/
│   └── style.css       # Complete styles (glassmorphism dark theme)
├── js/
│   ├── script.js       # Main JS (interactions, animations, UI logic)
│   └── ajax.js         # AJAX module (JSONPlaceholder API integration)
├── images/             # Add your profile image here
├── assets/             # Other static assets
└── README.md
```

## 🛠️ Tech Stack
- **HTML5** – Semantic structure
- **CSS3** – Glassmorphism, grid, flexbox, animations, responsive
- **JavaScript / jQuery** – DOM, events, animations, AJAX
- **AJAX** – Fetches posts from JSONPlaceholder API
- **Typed.js** – Hero typing animation
- **Particles.js** – Animated particle background
- **AOS** – Scroll-reveal animations
- **Font Awesome** – Icons
- **Formspree** – Contact form email integration
- **Google Maps Embed** – Location map

## ✨ Features
- 🌙 Dark / Light mode toggle with localStorage persistence
- 🖱️ Custom glow cursor
- ⚡ Loading screen animation
- 🎯 Animated skill progress bars
- 🔢 Counter animations
- 📡 AJAX-powered dynamic content from JSONPlaceholder API
- 📋 Form validation with jQuery
- 🗺️ Google Maps embed
- 📱 Fully responsive (mobile, tablet, desktop)
- 🍔 Hamburger mobile navigation
- 🔝 Scroll-to-top button
- 🔔 Toast notifications
- 💫 Floating animated social icons
- ✨ Particle.js animated background
- 🎨 Glassmorphism cards with hover glow effects
- ✍️ Typed.js hero typing animation

## 📦 How to Run
Simply open `index.html` in any modern web browser. No build process required.

For contact form, replace the Formspree endpoint with your own form ID from https://formspree.io
